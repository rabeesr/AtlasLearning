# Robotics Raw Sources

This directory is the holding area for source materials that have not yet been fully normalized into app-ready curriculum content.

## Conventions

- Put raw files in `assets/`.
- Add or update a manifest in `manifests/` for every new source.
- If a new source type or structure cannot be handled by an existing parser, record that explicitly before extraction work starts.
- Link each raw source to one or more topic slugs so content gaps stay visible in the curriculum tracker.
