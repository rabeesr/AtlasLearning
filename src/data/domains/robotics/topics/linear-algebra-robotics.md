# Why this topic matters

Robotics is full of quantities that live in space: positions, velocities, orientations, sensor frames, and transforms between them. Linear algebra gives you the language to describe those quantities precisely.

## In practice

- A robot arm joint configuration becomes a set of transformations.
- A camera observation is expressed in one frame and converted into another.
- A planner reasons about states that are often represented as vectors.

## What comes next

This topic unlocks forward kinematics, state estimation, and much of modern robotics math.
